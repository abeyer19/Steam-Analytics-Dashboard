# DVA-Project
Repo for DVA Project SP26 Team 40


## 1. Cloning Repo
To have the code dependencies function properly, you must have Python 3.14 installed, or higher.

Start by cloning the repo using your CLI.
#### Bash - Clone Repo: 
```Bash
git clone <GITHUB URL HERE>
```
\
Once cloned, either open the folder where to project has been cloned in your IDE of choice or change directories to the respective path and download depedencies. \
**It is highly recommended to create a virtual environment prior to downloading dependencies.**


#### Bash - Install Dependencies:
> a. Install dependencies.
```Bash
pip install .
```
> b. Install dependencies in editable mode for development.
```Bash
pip install -e .
```
